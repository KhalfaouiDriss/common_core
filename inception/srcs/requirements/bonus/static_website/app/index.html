About This Project

Inception is a system administration project from the 42 curriculum that challenges students to build a complete Docker-based infrastructure from scratch.

The project demonstrates containerization principles, networking, volume management, and security best practices using Docker and Docker Compose.

Technologies Used

Docker Debian NGINX WordPress PHP-FPM MariaDB Redis OpenSSL
8 Containers
1 Network
2 Volumes
