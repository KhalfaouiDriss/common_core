Three steps, one file

How it works

Everything happens between your browser and whichever API instance you configure. This page just brokers the conversation.

01

Point it at an API

Open Settings once and add the base URL of a cobalt-compatible instance — your own, or one you trust.

02

Paste the link

Drop in a YouTube, TikTok, Instagram, or Facebook URL and hit Get link.

03

Preview, then keep it

Watch the preview right here, then download straight to your device. No file ever passes through us.

Velto is a static page — it has no server and stores nothing. The API instance you configure does the actual fetching; its reliability, privacy practices, and platform support are outside this page's control. Only download videos you own or have permission to save, and respect the terms of the sites you paste links from. Brand names mentioned belong to their respective owners.
